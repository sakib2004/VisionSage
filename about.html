About VisionSage

VisionSage is a cutting-edge AI platform designed to enhance public safety through advanced image analysis. Developed at North South University, Dhaka, Bangladesh, our solution leverages state-of-the-art AI models to detect harmful objects, analyze scenes, and extract metadata from images, empowering law enforcement and security professionals.

Our Mission

We aim to create a safer world by providing intelligent, reliable, and accessible tools for real-time image analysis and threat detection.

Key Features

  • AI-powered harmful object detection (e.g., knives, guns)
  • Scene context analysis for situational awareness
  • Metadata extraction for forensic investigations
  • User-friendly interface with secure authentication
